undercover-dumper

Java agent that hooks class loading and writes .class files to disk.

Download ClassDumpAgent.jar Source

Setup

  1. Download and extract the release.
  2. Run setup.bat or copy a JVM line from below.
  3. Replace MOD_DUMPER_DIR with the install path.
  4. Paste into your launcher profile under Custom JVM Arguments.
  5. Launch Minecraft with the mod loaded.
  6. Output appears in dumped_classes/.
  7. Run pack_dump.bat to build dumped.jar.

JVM arguments

-javaagent:MOD_DUMPER_DIR\ClassDumpAgent.jar=MOD_DUMPER_DIR\dumped_classes
-javaagent:MOD_DUMPER_DIR\ClassDumpAgent.jar=MOD_DUMPER_DIR\dumped_classes|include:com/yourmod,dev/author
-javaagent:MOD_DUMPER_DIR\ClassDumpAgent.jar=MOD_DUMPER_DIR\dumped_classes|mode:all
-javaagent:MOD_DUMPER_DIR\ClassDumpAgent.jar=MOD_DUMPER_DIR\dumped_classes|mode:obfuscated

Modes

ModeFlagDumps
moddefaultobfuscated runtime names + include prefixes
obfuscatedmode:obfuscatedshort/hash-like names
includemode:includeinclude prefixes only
allmode:allnon-excluded classes

Options after the output path, separated by |: include:, exclude:, mode:

Files

  • ClassDumpAgent.jar - agent
  • build.bat - compile from source
  • setup.bat - print JVM args
  • pack_dump.bat - pack dump to jar
